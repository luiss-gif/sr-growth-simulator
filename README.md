#  Growth Simulator v6 取扱説明書

## 概要
このツールは、アプリ事業の成長シナリオをシミュレーションするための、単一HTML・ブラウザ完結型の双方向ダッシュボードです。最新のReactとTailwind CSSを駆使し、広告投資のユニットエコノミクス（LTV/CAC）をリアルタイムで可視化します。

---

## 主な機能

### 1. 実績データの活用
- **過去実績の確認**: 月ごとの広告費、DL数、登録数、CPI、CPAなどの実績を一覧表示。
- **Case 1へのコピー**: 実績セクション右上の「実績をCase 1にコピー」ボタンを押すことで、現在の数値をシミュレーションのベース（起点）として即座に反映できます。

### 2. 成長シナリオ比較（3案同時比較）
- **Case 1 (現状維持)** / **Case 2 (改善案)** / **Case 3 (理想案)**
- 各カード内の「レバーを編集」ボタンから、以下の変数をその場で調整可能。
  - 広告予算、CPI
  - 登録率、チュートリアル完了率、課金転換率
  - ARPU（客単価）、継続期間

### 3. KPI・専門用語の定義（ツールチップ）
各ラベル横の「ⓘ」アイコン（アイコン・ラベル上をhover/タップ）で詳細な定義が表示されます。

- **LTV/CAC**: 顧客生涯価値 ÷ 獲得コスト。2.0以上が健全な目安。
- **1人当たり利益**: LTV − CAC。黒字なら緑、赤字なら赤で表示。
- **月次売上（MRR）**: 月次のサブスク売上（課金ユーザー数 × ARPU）。
- **回収期間**: CACを回収するのに必要な期間（CAC ÷ ARPU）。

### 4. コンバージョン・ファネル
- `インストール` → `ユーザー登録` → `チュート完了` → `有料課金` の流れをバーチャートで表示。
- どこがボトルネック（最大の離脱ポイント）になっているかを視覚的に特定できます。

### 5. 通期成長予測・アドバイス
- 今後1年間の売上・登録者数の積み上げ予測をグラフ化。
- シミュレーション結果に基づき「通期赤字の警告」や「投資拡大の推奨」といったAIアドバイスを表示。

---

## 推奨ワークフロー
1. **ベースライン設定**: 「実績をCase 1にコピー」する。
2. **変数調整**: 「レバーを編集」し、CPIの変動や転換率の改善が「1人当たり利益」にどう響くかを確認。
3. **意思決定**: 3つのケースを比較し、目標とすべき必達KPIレベル（ユニット利益のプラス化）を合意する。
4. **報告・共有**: 「PDFエクスポート」を使用して資料化。
